Facebook LBPP adds the new facebook likebox to your website through widget and shortcode

== Description ==

A small plugin to add the new Facebook Like Box to widgets and content, all the options are customizable in the settings area on the admin panel

Features in Facebook LBPP:

* Can be used as a shortcode
* All options of the plugin are customizable
* Has a widget

== Installation ==

Upload the Facebook LBPP plugin to your blog, Activate it, go to Facebook LBPP Settings to enter your FB page url ,and you're good to go.
